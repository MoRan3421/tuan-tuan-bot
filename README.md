## 🌟 核心特性 (Key Features)

- **🧠 双核 AI 引擎**: 集成 Google Gemini 2.0 与 Groq Llama 3.3，提供极速、智能的熊猫灵魂对话。
- **🎵 高清音乐电台**: 支持 YouTube、Spotify、SoundCloud 全平台点歌，极致音质，秒速加。
- **🛡️ 服务器时光机**: 完整的服务器备份与恢复系统，守护每一份珍贵的社区回忆。
- **🎮 熊猫乐园 (800+ 指令)**: 包含钓鱼、结婚、转盘、猜拳、30+ 种萌萌社交互动及百科知识库。
- **🌐 Elite Hub 控制台**: 现代化的 React 网页后台，支持实时修改机器人样貌、切换 AI引擎、管理功能开关。
- **💎 至尊特权系统**: 完善的 Freemium/Premium 体系，集成 Stripe 支付，支持月付与永久买断。
- **📱 移动端 PWA 支持**: 网页支持手机安装为 App，随时随地管理机器人。

## 🚀 快速开始 (Quick Start)

### 机器人核心 (Bot Core)
1. `cd bot`
2. `npm install`
3. 配置 `.env` 文件 (填入 Discord Token, Firebase Key, AI API Keys)。
4. `npm start`

### 网页后台 (Elite Hub)
1. `cd web`
2. `npm install`
3. `npm run build`
4. 部署至 Firebase Hosting: `firebase deploy`

## 🛠️ 技术栈 (Tech Stack)

- **Frontend**: React 19, Vite, TailwindCSS, Framer Motion, Lucide-React.
- **Backend**: Node.js, Express, Discord.js v14, Discord-Player.
- **Database**: Firebase Firestore.
- **Auth**: Firebase Auth (Discord OAuth2 Provider).
- **Payment**: Stripe API.
