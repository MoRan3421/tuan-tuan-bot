## 🚀 快速开始 (Quick Start)

### 🤖 机器人核心 (Bot Core)
```bash
cd bot
npm install
# 配置 .env 文件 (填入 Discord Token, Firebase Key, AI API Keys)
npm start
```

### 🌐 网页后台 (Elite Hub)
```bash
cd web
npm install
npm run dev    # 开发模式
npm run build  # 生产构建
firebase deploy # 部署到 Firebase Hosting
```
