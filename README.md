## ✨ 至尊核心特权

### 🧠 双核 AI 大脑 (Supreme AI)
*   **Gemini 1.5 极速引擎**：拥有深厚的知识库，不仅懂游戏，更懂主人的心。
*   **私信互动**：支持 Direct Message 对话，您的秘密只有团团知道。
*   **语义识别**：不需要死板的指令，直接艾特它就能开启畅聊。

### 🎶 高保真云端乐厅 (Hifi Music)
*   支持 **YouTube / Spotify / Soundcloud** 全平台解锁。
*   24bit 采样率，即便是云端播放也拥有 CD 级的质感。
*   支持歌词同步显示，开启属于您的直播间背景乐。

### 📊 熊猫等级与经济 (Economy V3)
*   **至尊排行卡 (Rank Card)**：全客制化卡片，展示您的竹子存量与成长等级。
*   **竹林大冒险**：通过 `/daily` 和 `/work` 获取竹子，在 `/leaderboard` 上挑战至尊地位。

## 🚀 快速上手 (Quick Start)

1.  **邀请团团**：点击 [官方邀请链接](https://tuantuanbot-28647.web.app) 将团团带往您的家。

## 🛠️ 技术栈 (Tech Stack)
*   **Core**: Node.js 22 + Discord.js v14
*   **Frontend**: React + Vite + TailwindCSS
*   **Mobile**: Flutter (iOS / Android Sync)
*   **Cloud**: Google Cloud Run + Firebase + GitHub Actions

## 🛠️ 至尊维护指南 (Supreme Maintenance)

如果云端部署提示 **Permission Denied**，请使用内置的至尊修复脚本：

1.  **云端修复 (推荐)**：
    *   在 Google Cloud Shell 中执行：`bash gcp-fix/fix.sh`
2.  **本地修复**：
    *   在 PowerShell 中执行：`.\gcp-fix\fix.ps1`
